For the modern university student, there are four essentials for survival: air, food, water, and an internet connection. Without reliable connectivity, most academic activities simply cannot proceed. This guide will walk you through how to access all of Lingnan University's network services, ensuring you stay connected both on and off campus.


## Campus Wi-Fi Network Guide

Lingnan University provides five Wi-Fi options to meet your networking needs both on campus and across other institutions.

### Wi-Fi Network Overview

| SSID | Login Method |
| :--- | :--- |
| **LU** | Use your myLingnan Portal username and password |
| **LU-Advanced** | Use your myLingnan Portal username and password (offers 5G high-speed connection) |
| **Wi-Fi.HK via LU** | No password required; click "Confirm" on the login page after connecting |
| **eduroam** | **Username:** Full university email; **Password:** myLingnan Portal password |
| **University via CSL** | **Username:** Full university email; **Password:** myLingnan Portal password |

### Device Connection Tips

| Operating System | Setup Instructions |
| :--- | :--- |
| **Android** | Select **PEAP** as the EAP method and set CA certificate to **"Don't validate"**. |
| **iOS** | After connecting, enter your credentials and tap **"Trust/Accept"** when the certificate prompt appears. |
| **Windows / macOS** | A login window will pop up automatically; simply enter your myLingnan Portal credentials. |

> **Pro Tip:** `eduroam` is available at participating universities worldwide; `University via CSL` is primarily deployed across Hong Kong university campuses, making it perfect for cross-institutional study.

## Campus VPN (Essential for Off-Campus Access)

Certain key administrative resources (such as accessing Course Outlines, the Counselling Services Booking System, and applying for courses that are already full) are restricted to the campus internal network. When you are off campus, follow these steps to use the VPN:

1.  **Install the App:** Download **F5 Big-IP Edge Client** or **F5 Access** from the official [ITSC VPN webpage](https://www.ln.edu.hk/cht/itsc/services/network/vpn-connection).
2.  **Server Setup:** Open the app, enter `https://studentvpn2.ln.edu.hk` as the server address, and click **"Connect"**.
3.  **Authentication:** Enter your login credentials and complete the **2FA (Two-Factor Authentication)**.
4.  **Manage Connection:** Always click **"Disconnect"** once you have finished your tasks.

> Note: Using a VPN may slightly reduce your internet speed, so it is recommended to enable it only when accessing restricted campus resources.

## Dormitory Wired Network (Wired Network)

If you are using a computer in the student hostel, the wired network provides a more stable connection:

1.  Plug your computer into the wall-mounted network port.
2.  Open a web browser and try to visit any website; the system will automatically redirect you to the login page. If it does not redirect, try visiting the [university homepage](https://www.ln.edu.hk).
3.  Log in using your **myLingnan Portal** credentials.

### Important Regulations
*   **No Private Networking Equipment:** Installing your own routers, range extenders/repeaters, or network-attached storage (NAS) devices in the dormitories is strictly prohibited.
*   **Device Restrictions:** The wired network is intended for computers with a web browser. Devices without a browser interface—such as network printers, IP cameras, or TV boxes—are not supported.
