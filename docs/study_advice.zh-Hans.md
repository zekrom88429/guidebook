关键机制是让你的大脑进行「挣扎」：

1. 用布鲁姆认知目标分类学（Bloom's revised taxonomy）中至少第 3 级的程度来解释概念，而不是从底层开始。
2. 用量化指标来评估你的理解程度，这在当今的 AI 时代很容易做到。
3. 用类比法来理解概念。
4. 使用 Anki 等应用程式对术语进行间隔重複（Spaced Repetitions）。
