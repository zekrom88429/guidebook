大學生生存有四大條件，空氣、食物、水和互聯網連接。沒有互聯網連接，大部份學習活動也無法展開。希望這一份指南將帶你掌握嶺南大學校內所有網絡服務的連接方式，讓你在校園內外都能隨時保持連線狀態！

## 校園 Wi-Fi 網絡指南

嶺南大學提供五種 Wi-Fi 選項，以滿足你在校內及跨校的網絡需求。

### Wi-Fi 網絡概覽

| SSID | 登入方式 |
| :--- | :--- |
| **LU** | 使用 myLingnan Portal 的用戶名稱與密碼 |
| **LU-Advanced** | 使用 myLingnan Portal 的用戶名稱與密碼 (提供 5G 高速連線) |
| **Wi-Fi.HK via LU** | 無需密碼；連接後於瀏覽器確認即可使用 |
| **eduroam** | **用戶名：** 完整大學電郵；**密碼：** myLingnan Portal 密碼 |
| **University via CSL** | **用戶名：** 完整大學電郵；**密碼：** myLingnan Portal 密碼 |

### 裝置連線設定要點

| 作業系統 | 設定技巧 |
| :--- | :--- |
| **Android** | EAP 方式選擇 **PEAP**，CA 憑證設為 **「不驗證」**。 |
| **iOS** | 連線後輸入帳號密碼，系統跳出憑證時點選 **「接受」** 即可。 |
| **Windows 和 macOS** | 系統會自動彈出驗證視窗，輸入 myLingnan Portal 帳號密碼即可。 |

> `eduroam` 可在全球各地參與的大專院校使用；`University via CSL` 則主要部署於香港各大學校園，是跨校學習的幫手。

## 校園 VPN 連線（校外存取必備）

部分關鍵校務資源（常用的有查閱課程大綱、預約心理諮詢系統及申請額滿課程），僅限校園內網存取。當你身處校外時，請依照下列步驟連線 VPN：

1.  **安裝程式：** 於學校[VPN 連接網頁](https://www.ln.edu.hk/cht/itsc/services/network/vpn-connection) 下載 **F5 Big-IP Edge Client** 或 **F5 Access**。
2.  **設定伺服器：** 伺服器網址請輸入 `https://studentvpn2.ln.edu.hk`，點擊 **Connect**。
3.  **身份驗證：** 輸入帳密並完成 **2FA (雙重認證)**。
4.  **管理連線：** 使用完畢後，請務必點擊 **Disconnect**。

> 注意：VPN 連線會略微降低網速，建議僅在存取校內資源時開啟。

## 宿舍有線網絡服務 (Wired Network)

若你在宿舍使用電腦，有線網絡提供更穩定的連線品質：

1.  將電腦連接至網絡接口。
2.  開啟瀏覽器並嘗試瀏覽網頁，系統將自動導向認證頁面。若未轉跳，請直接訪問 [學校首頁](https://www.ln.edu.hk)。
3.  以 **myLingnan Portal** 的帳號密碼完成登入。

### 重要使用規範
*   **禁止私設設備：** 宿舍內嚴禁安裝路由器 (Router)、網絡中繼器 (Repeater) 或網絡儲存伺服器 (NAS)。
*   **裝置限制：** 有線網絡僅供具備瀏覽器功能的電腦使用。不支援無瀏覽器介面的設備，如網絡印表機、IP 攝影機或電視盒子等。
