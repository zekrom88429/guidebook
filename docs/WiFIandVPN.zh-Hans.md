大学生生存有四大条件，空气、食物、水和互联网连接。没有互联网连接，大部份学习活动也无法展开。希望这一份指南将带你掌握岭南大学校内所有网络服务的连接方式，让你在校园内外都能随时保持连线状态！

## 校园 Wi-Fi 网络指南

岭南大学提供五种 Wi-Fi 选项，以满足你在校内及跨校的网络需求。

### Wi-Fi 网络概览

| SSID | 登入方式 |
| :--- | :--- |
| **LU** | 使用 myLingnan Portal 的用户名称与密码 |
| **LU-Advanced** | 使用 myLingnan Portal 的用户名称与密码 (提供 5G 高速连线) |
| **Wi-Fi.HK via LU** | 无需密码；连接后于浏览器确认即可使用 |
| **eduroam** | **用户名：** 完整大学电邮；**密码：** myLingnan Portal 密码 |
| **University via CSL** | **用户名：** 完整大学电邮；**密码：** myLingnan Portal 密码 |

### 装置连线设定要点

| 作业系统 | 设定技巧 |
| :--- | :--- |
| **Android** | EAP 方式选择 **PEAP**，CA 凭证设为 **「不验证」**。 |
| **iOS** | 连线后输入帐号密码，系统跳出凭证时点选 **「接受」** 即可。 |
| **Windows 和 macOS** | 系统会自动弹出验证视窗，输入 myLingnan Portal 帐号密码即可。 |

> `eduroam` 可在全球各地参与的大专院校使用；`University via CSL` 则主要部署于香港各大学校园，是跨校学习的帮手。

## 校园 VPN 连线（校外存取必备）

部分关键校务资源（常用的有查阅课程大纲、预约心理谘询系统 及申请额满课程），仅限校园内网存取。当你身处校外时，请依照下列步骤连线 VPN：

1.  **安装程式：** 于学校[VPN 连接网页](https://www.ln.edu.hk/cht/itsc/services/network/vpn-connection) 下载 **F5 Big-IP Edge Client** 或 **F5 Access**。
2.  **设定伺服器：** 伺服器网址请输入 `https://studentvpn2.ln.edu.hk`，点击 **Connect**。
3.  **身份验证：** 输入帐密并完成 **2FA (双重认证)**。
4.  **管理连线：** 使用完毕后，请务必点击 **Disconnect**。

> 注意：VPN 连线会略微降低网速，建议仅在存取校内资源时开启。

## 宿舍有线网络服务 (Wired Network)

若你在宿舍使用电脑，有线网络提供更稳定的连线品质：

1.  将电脑连接至网络接口。
2.  开启浏览器并尝试浏览网页，系统将自动导向认证页面。若未转跳，请直接访问 [学校首页](https://www.ln.edu.hk)。
3.  以 **myLingnan Portal** 的帐号密码完成登入。

### 重要使用规范
*   **禁止私设设备：** 宿舍内严禁安装路由器 (Router)、网络中继器 (Repeater) 或网络储存伺服器 (NAS)。
*   **装置限制：** 有线网络仅供具备浏览器功能的电脑使用。不支援无浏览器介面的设备，如网络印表机、IP 摄影机或电视盒子等。
